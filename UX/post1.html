Action vs Information

Introduction

A typical user interface has many different types components,some of them "do stuff" other are just there to give information about something.We have learned to discern different types of elements within the UI. For example a UI with an 'X' inscribed or represented somehow is known to have the action of closing or deleting something.On the other hand a number such as 07:34 would almost certainly be something related to time.

In these examples we could deduce that a typical user understands the difference between actionable elements and informational elements
